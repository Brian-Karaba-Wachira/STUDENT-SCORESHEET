# STUDENT-SCORESHEET
A simple Java console program that generates a beautiful, perfectly aligned school grade report just like professional ones!

Features:  
- Takes input for school name, teacher, grade, year & number of students  
- Accepts each student's name and marks in 6 subjects (English, Math, History, Geography, Science, Programming)  
- Automatically calculates total marks, letter rank (A/B/C/D/F), subject totals & averages  
- Displays everything in a clean, tabulated format with perfect column alignment  
- Shows final rank summary (A's, B's, C's, D's, F's count)
